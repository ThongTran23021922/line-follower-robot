\# Line Follower Robot



\## Description

A line following robot using IR sensors and microcontroller.



\## Features

\- PID control

\- Line tracking

\- Left / Right turn detection

\- Finish line detection

\- Real-time mapping



\## Hardware

\- STM32

\- IR Sensor Array

\- TB6612 / L298

\- DC Motors
